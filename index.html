Profile

因緣際會

早在起初從沒想過自己會去寫程式，從前對程式的印象只有晦澀難解 ，但卻從不失對程式的好奇，畢竟是個愛打電玩的人，總是會好奇著遊戲是怎麼完成的 ，而在某次因緣際會下，大學的同學帶我去黑客松，起初我只是衝著免費披薩跟飲料去的 ，想不到在那裏聽了一些講座之後，突然發現好像寫 code 也沒有想像中這麼難， 而且格外有趣，便一頭栽進了程式的世界。

越熟知，越無知

賈伯斯曾說 Stay hungry. Stay foolish. 這句話的含意我始終不是很能體會，直到最近。 在剛踏入前端時，會寫幾個 html、會用 css 切版、稍微會用 jQuery 做一些簡單的動畫，就自詡為前端工程師了。 然而，在這時期我所接觸到的前端也只是冰山一角，但是卻遇到了 一位朋友，他簡單的告訴了我一些前端當今時下正紅的技術， React、vue.js、bootstrap、webpack 等等... 我聽得滿頭霧水，才了解到自己是井底之蛙。

海納百川

在認識到自己的無知之後便想努力補足自己的缺失，因此努力學習各種技術， 不僅僅是前端，從設計 UI / UX 到後端，甚至是一些的環境，能學的都學 能吸收的都吸收。看了一些設計規範以及 UI / UX 觀念、一些 webpack 相關 知識、SSR 以及 prerender 、Node.js(express)、PHP + apache2 等等... 不敢說精但希望至少在遇到其他相關領域上不要是一知半解，甚至可以完全了解。

Experience

 2015 - 2017 寰宇數理教育中心

大學時期為了幫自己多賺點零用錢買想要的電玩設備所以跑去打工，然而這一做下去就不知不覺變成了組長，從一個小小的打工仔變成要管一大群打工仔的小組長，從本來只是一個聽從指示做東做西變成要做決定的人，而這一做也做了兩年半，當中學到最重要的是[溝通]，因為組長這個位子對下要傳達上頭給的指示以外還要糾正錯誤要跟他們溝通，而對上需要反饋下面遇到的困難而做一些建議進而改進，以及對家長要說明學員可能遇到了甚麼問題希望有甚麼解決方式也要跟家長溝通。再來就是教學經驗的提升了，某種程度就是更能夠站在別人的角度去思考，因為要用學生的思考模式才有辦法教懂學生。

2017 畢業於 國立台北科技大學

大學之中雖然念的是機械系，然而，卻因為對於設計以及程式很有興趣，自學了設計、影像後製相關學得有些許心得也做出了一些作品；也自學了 swift 但這就比較撞牆畢竟當時 swift 算是新東西資訊也不是很多，而有一位同學也是自學寫程式聽到我也在學便邀請我一起去黑客松，稍稍啟發了我網頁工程這一塊。我也繼續鑽研甚至在專題研究上連教授也不懂以致沒人詢問的狀況下用 HTML、 JS、 jQuery 自製了一個簡易的打鼓系統。然後在網路上接觸了更多資訊，也知道我所學到的只不過世滄海一粟，但我也沒有因此怯懦，只知道既然要成為一個 coding 人就肯定要永無止盡的學習。

2018 - now 采億科技股份有限公司

在這裡算是讓我成長最大量的部分，雖然主產品的部分是以 JAVA 為後端並搭配 Spring Boot 為框架 ，而前端的部分基本都是 jQuery 作為 library 而未使用框架，但大部分其他的專案基本都是以 vue.js 進行開發的，而我所參與的專案也基本上全是用 vue 進行開發，也有我一人獨立完成的 vue cli 3 的專案 ( 公司官網 )，因此對 vue 的操作上十分熟悉。

也因為一些需求，常常會使用到 CSS 3 的 animation 去繪製動畫，更能搭配 scss / sass 去繪製較複雜的 動畫，對於 css 以及 scss / sass 的熟悉度也算是滿高的。
